# NCScope 고지

NCScope는 공공기관 채용 실무를 보조하기 위한 비공식 오픈소스 프로젝트입니다.

## 공식 서비스와의 관계

- NCScope는 공식 NCS 사이트 또는 한국산업인력공단의 공식 서비스가 아닙니다.
- NCScope는 공식 NCS 로고, 아이콘, 이미지, 화면 구성, 브랜드 자산을 포함하거나 복제하지 않습니다.
- 화면 색상과 정보 구조는 공공서비스형 업무 도구에 맞춰 독자적으로 설계했습니다.
- NCS, 국가직무능력표준 등 관련 명칭과 상표의 권리는 각 권리자에게 있습니다.

## 데이터 사용

- NCScope 앱 저장소에는 원본 12.6GB NCS DB, `NCS_DB.xlsx`, 로컬 SQLite DB를 포함하지 않습니다.
- NCS 능력단위·수행준거·KSA 조회는 별도 read-only NCS_MCP serving DB를 통해 수행합니다.
- serving DB 배포·운영 시 원천 데이터의 이용 조건, 공공데이터 라이선스, 기관 내부 보안 기준을 별도로 확인해야 합니다.

## 면접 운영 책임

- NCScope가 생성한 질문은 초안입니다.
- 실제 채용 절차에 사용하기 전 담당자가 세분류, 능력단위, KSA 근거, 차별·민감 질문 여부를 최종 검토해야 합니다.
- 세분류가 정확히 매칭되지 않는 경우 NCScope는 자동 생성을 중단하고 수동 선택 흐름으로 전환합니다.
